Description

This is the dataset used in the ICRA'24 paper "Direct 3D model-based object tracking with event camera by motion interpolation". The dataset consists of some trajectories of an event camera (stored as a rosbag) and corresponding 3D models in the form of point cloud. The dataset contains 4 different objects, Cracker Box, Bleach Cleanser, Xbox Controller and Cup Noodles. All trajectories were captured using a Prophesee Gen3.1 camera with a 8mm pinhole lens. The camera resolution is 640x480. The calibration data is provided as well.
The rosbag files contain the events using /prophesee/camera/cd_events_buffer message types.
